Points I'd like my future self to remember.

  1. Memoization refers to saving already computed values in a globally accesible place. This can save a lot of computational resources whenever some computation needs to be repeated again and again in an algorithm. Make memos when you can. (Source: Kleinberg-Tardos, chapter on dynamic programming.)
